Overview

This project develops a full pipeline for the classification of cancer cells using k-Nearest Neighbors (k-NN), distinguishing benign from malignant samples. It provides robust tools for data cleaning, preprocessing, validation, and metric visualization.

The Random Forest model achieved the best performance with 97.3% accuracy.

Implementation Overview

The project uses multiple algorithms and a small neural network to recognize a set of rehabilitation exercises. Each algorithm’s performance was compared to determine the most suitable model for this application. A comprehensive preprocessing phase was also carried out on the original dataset.
