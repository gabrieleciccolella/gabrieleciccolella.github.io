Project Overview

The main goal of this project is to build a comprehensive machine learning pipeline for the classification of cancer cells, aiming to distinguish between benign and malignant samples. Using a combination of advanced data preprocessing and multiple validation strategies (Holdout, Random Subsampling, and Stratified Shuffle Split), the model ensures flexibility, robustness, and interpretability of results.

In the context of oncology, accurately distinguishing between benign and malignant cells is critical for early and reliable diagnosis. The dataset used in this study includes key cell-based metrics and indices that serve as important diagnostic features. This pipeline aims to offer a practical and data-driven decision-support tool for professionals in the medical field.

Validation Strategies

  • Holdout: A classic split where the user selects the training percentage (e.g., 80%), and the rest is used for testing.
  • Random Subsampling: Multiple random splits of the dataset. The user defines the number of iterations and training/test proportions.
  • Stratified Validation: A variation that maintains class balance (benign/malignant) in each split, ideal for imbalanced datasets.

Each strategy offers a different way to evaluate the model’s generalization and stability.

Evaluation Metrics

Once the model evaluation is complete, several key metrics are automatically computed to provide a comprehensive view of performance:

  • Accuracy: Percentage of correctly classified samples.
  • Error Rate: Percentage of misclassified samples.
  • Sensitivity (Recall): Ability to correctly identify malignant cells.
  • Specificity: Ability to correctly identify benign cells.
  • Geometric Mean: Balances Sensitivity and Specificity for a more stable evaluation.

These metrics together offer a holistic view of model behavior, helping identify both strengths and weaknesses under different validation setups.
