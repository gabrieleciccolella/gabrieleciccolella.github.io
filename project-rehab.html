Overview

I developed several machine learning algorithms and a small neural network to recognize a set of rehabilitation exercises. I compared their performance to identify the most suitable model for this application.

  • Algorithms: LDA, Decision Tree, Logistic Regression, SVM, KNN, Random Forest, Gaussian Naive Bayes, AdaBoost.
  • Metric: Accuracy (%), evaluated via consistent splits and identical preprocessing across models.

Results (Accuracy)

Best model highlighted. Hover bars for exact values.

Key Takeaways

Random Forest achieved the best performance (97.30%), followed closely by Logistic Regression (96.96%) and LDA (93.02%), making ensemble and linear models particularly effective for this dataset.
