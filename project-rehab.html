Thesis Overview

This project represents my Bachelor’s thesis in Industrial Engineering and focuses on the development of a machine learning system for recognizing rehabilitation exercises. The research aimed to analyze and compare multiple supervised learning algorithms applied to human motion data.

A set of physiotherapeutic exercises was selected and performed by a large number of patients under standardized conditions. From these sessions, a structured dataset was created to train and evaluate several classification models capable of identifying which exercise was being executed.

The study demonstrates how intelligent algorithms can support physiotherapists by automatically detecting exercise type and execution quality, paving the way for smart rehabilitation monitoring tools.

Methodology

  • Dataset: Constructed from kinematic data collected across multiple rehabilitation sessions.
  • Preprocessing: Data cleaning, normalization, and feature scaling to ensure consistent input quality.
  • Models tested: LDA, Decision Tree, Logistic Regression, SVM, KNN, Random Forest, Gaussian NB, AdaBoost.
  • Evaluation metric: Accuracy (%) through repeated validation with identical preprocessing pipelines.

Conclusions

The comparison of models demonstrated that ensemble and linear approaches yield the best generalization performance. Random Forest achieved top accuracy, showing robustness against noise and variability in patient motion.

This thesis highlights how machine learning can enhance physiotherapy by offering reliable, data-driven support for rehabilitation assessment and progress tracking.
